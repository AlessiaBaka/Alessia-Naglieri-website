Javascript art

Generative images with processing

Processing and javascript

These images have been computer-generated by writing a javascript code using the app Processing. Javascript is a powerful programming language that I've used to explore generative images creation. In this case, starting from a given set of parameters, the code generates iteratively an image. Therefore, tuning the initial parameters one can get different images.

/*Processing*/, the book

I've used generative imaging to create the cover of this Processing guidebook. This cover resembles a kaleidoscope, symbolizing the creative power that an artist can express using this tool.
